Cookies

Ce script vous permet de créer des cookies simplement grâce à la Prototype.js, donc celui-ci est indispensable.

Créer un cookie


			var mail = new Cookies();

			...

			mail.set('mailSend'x, 'yes', 1);

Le code ci-dessus crée un cookie ayant pour clé "mailSend", pour valeur "yes" et pour durée de validité 1 jour.

Récupérer la valeur d'un cookie


			var value = mail.get('mailIsSend');

Le code ci-dessus récupère la valeur du cookie créé ci-dessus. La fonction get renvoie false si le cookie recherché n'existe pas, il est donc très facile de tester la présence du cookie.

Supprimer un ou plusieurs cookies


			mail.clear('mailSend');

			mail.clearAll();

La première ligne du code ci-dessus supprime le cookie ayant pour clé "mailsend".

La deuxième ligne du code ci-dessus supprime tous les cookies.
