Onglet en javascript

Ce script vous permet de construire des onglets simplement. Il est basé sur Prototype.js, donc celui-ci est indispensable. Il vous suffit de créer sur votre page HTML une liste et autant de panel qu'il y a d'élément dans la liste.

		<ul id="onglets">
			<li idPanel="un"> Un </li>
			<li idPanel="deux"> Deux </li>
			<li idPanel="trois"> Trois </li>
			<li idPanel="quatre"> Quatre </li>
			<li idPanel="cinq"> Cinq </li>
		</ul>
		
		<div id="un"> Onglet 1 </div>
		<div id="deux"> Onglet 2 </div>
		<div id="trois">
			<img width="240" src="../images/chat.jpg"/>
		</div>
		<div id="quatre"> Onglet 4 </div>
		<div id="cinq"> Onglet 5 </div>

	
		window.onload = function()
		{	
			...
			new Tabs('onglets');
			...
		}

Chaque élément de la liste doit avoir un attribut 'idPanel' qui correspond à l'id du panel qui lui est associè. De plus chaque élément de la liste a une classe suivant son état : 'ongletActif' ou 'ongletPassif', et chaque panel a une classe 'panelTab'.
