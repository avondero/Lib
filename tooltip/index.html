Tooltip

Ce script permet de configurer un ou plusieurs tooltips sur n'importe quel élément de votre page HTML, et ceci de façon simple. Il est basé sur Prototype.js (= 1.6), donc celui-ci est indispensable. Tous les scripts suivant dans la page sont exécuter dans la fonction 'window.onload' 


			window.onload = function()

			{

				...

				new Tooltip('id', options);

				...

			}

Tooltip simple (sans option)


			new Tooltip('demoSimple',{

				content: "Démo simple"

				});

			...

		
	

			<span id="demoSimple">Démo</span>

Le code ci-dessus vous permet de construire un tooltip simple apparaissant en bas à gauche de l'objet survolé.Démo

Tooltip avec id et class


			#tooltipCSS

			{

				color: #00FFFF;

			}



			.tooltipCSSClass

			{

				background-color: #0000FF;

			}



		

			var content = "Lorem ipsum dolor [...] interdum auctor venenatis.";



			new Tooltip('demoCSS',{

				content: content,

				idTooltip: 'tooltipCSS',

				nameClass: 'tooltipCSSClass'

				});

			...

		
	

			<span id="demoCSS">tooltip</span>

Un id et une class peuvent être ajouter au tooltip pour lui attribuer un style personnalisé.

Tooltip avec transparence et largeur maximum


			var content = "Lorem ipsum dolor [...] interdum auctor venenatis.";



			...

			

			new Tooltip('demoOpacite',{

				width: 200,

				content: content,

				opacity: .5

				});

			new Tooltip('demoTailleMaxModif',{

				content: content,

				maxWidth: 400

				});

			...

		
	

			<span id="demoOpacite">message</span>

			<span id="demoTailleMaxModif">bulle</span>

Cette fois ci la première ligne du code précédent vous permet de changer l'opacité du message. Tandis que la deuxième vous permet de modifier la largeur maximum de la bulle qui est par défaut de 200px.

Tooltip ancré à différent endroit d'un élément


			tooltipHook = new Tooltip('demoHook',{

				content: "Démo hook",

				hook: {

					target: 'bottomLeft',

					tip: 'outside'

					}

				});

		

			<select id="selectHookTarget" 

				onchange="tooltipHook.setHook({target: $F(this)});" >...</select>

			<select id="selectHookTip" 

				onchange="tooltipHook.setHook({tip: $F(this)});">...</select>

			

			<img id="demoHook" src="../images/chat.jpg"/>

L'exemple ci-dessous parle de lui-même, vous avez 18 choix de position pour votre tooltip.


Activation/désactivation du tooltip


			tooltipAutoStart = new Tooltip('demoAutoStart',{

				content: "Démo simple",

				autoStart: false

				});

		

			<span id="demoAutoStart" >tooltip</span>

			

			<input type="button" value="Activer/Désactiver" 

				onclick="tooltipAutoStart.toggleTooltip()"/>

Cliquer sur le bouton ci-dessus pour activer/désactiver le tooltip. Le fonctions enable() et disable() existe.
